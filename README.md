React practice
==========

### Description

> A practical react project which is based on React and Redux.

### Prerequisite

> Ensure to have NodeJS v6.x MINIMUM.

### Install
```bash
npm install
```

or 

```bash
yarn install
```

### dev : hot reload mode

Development mode = best dev experience (*hot reload, redux devtools extension...*).

```bash
npm run start
```

> Then go to [localhost:3001]
